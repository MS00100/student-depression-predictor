# 🧠 Student Depression Predictor

A Machine Learning web app that predicts student depression risk using AdaBoost.

## 🚀 Features
- Interactive UI (Streamlit)
- Real-time prediction
- Automatic model training

## 🛠️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
